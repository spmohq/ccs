🟢 STAGE 3 — DESIGN YOUR PRODUCT

Now you turn that problem into a machine (prompt).

🎯 Formula: Action + Task + Audience + Output

ROBOT PROMPT: CREATION
Create a reusable ChatGPT prompt that helps [target audience] solve this problem: Problem: [PASTE PROBLEM] The prompt must: - Be easy to use (plug-and-play) - Produce actionable output - Save time - Include clear instructions Output format: 1.
